# quant-personal-tests

This repository serves as a comprehensive collection of quantitative trading strategies and modeling approaches I intend to test for spot trading in stocks, government bonds, commodities, and forex. It is not limited to fully functional or profitable strategies—that's perfectly fine. The primary goal is to build a structured repertoire of ideas for a major future project.

All code will be implemented in Jupyter notebooks using pure Python, with a strong emphasis on machine learning techniques such as classification and linear regression models.

## Key Categories Covered

The notebook will systematically explore these five core categories:

- **Correlation**: Identifying and exploiting statistical relationships between assets
- **Mean Reversion**: Strategies betting on price reversals to historical means
- **Momentum**: Trend-following approaches capturing persistence in price movements
- **Periodicity**: Time-based patterns like seasonality, intra-week cycles, or calendar effects
- **Risk Management**: Techniques for position sizing, volatility targeting, and portfolio optimization
